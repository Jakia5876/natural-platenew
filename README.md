# Natural Plate - Eco-Friendly & Biodegradable Tableware

Natural Plate is a premium e-commerce platform dedicated to providing sustainable, organic, and biodegradable tableware solutions. Our mission is to replace harmful plastics with natural alternatives like Areca leaf plates, ensuring a win for both the environment and your dining experience.

## ✨ Features

-   **Premium Storefront**: Beautifully designed homepage with category browsing.
-   **Extensive Catalog**: Wide range of Areca leaf plates (Round, Square, Deep, Snack) and organic grocery items.
-   **Seamless Cart & Checkout**: Intuitive shopping cart and multi-step checkout with a focus on "Cash on Delivery".
-   **Order Tracking**: Real-time order status tracking with a visual progress timeline.
-   **User Dashboard**: Personalized area for customers to view order history and manage profiles.
-   **Admin Panel (Mockup)**: Interface for managing sales, orders, and inventory.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

-   **Framework**: [Next.js (App Router)](https://nextjs.org)
-   **Styling**: Vanilla CSS Modules
-   **State Management**: React Context API
-   **Language**: TypeScript

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/natural-plate.git
    cd natural-plate
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Open the application**:
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser.
